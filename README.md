# 🧠 SelfDecode

SelfDecode is a lightweight web application that analyzes user responses to generate personality insights and behavioral patterns.

The application processes answers to structured prompts and produces interpretations based on response patterns.

---

## 🌐 Live Demo

Try the app here:

https://selfdecode.onrender.com/

---

## 💡 About the Project

SelfDecode explores how structured questions and responses can be used to generate insights about personality traits.

Users answer a set of prompts and the application evaluates their responses to provide a reflection of behavioral tendencies.

The project focuses on simplicity while demonstrating how a small backend system can power an interactive web experience.

---

## ✨ Features

- Interactive personality analysis
- Dynamic question-based responses
- Backend processing for insights
- Clean and minimal interface
- Instant results generation

## 🚀 Running Locally

Clone the repository

```
git clone https://github.com/wizard-dreamer/selfdecode.git
```

Navigate to the project folder

```
cd selfdecode
```

Install dependencies

```
pip install -r requirements.txt
```

Run the server

```
python server.py
```

Open in your browser

```
http://localhost:5000
```
